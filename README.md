\# Terraform Lab3



\## Опис проєкту

Цей проєкт демонструє базову конфігурацію Terraform для AWS з модульною структурою.



\## Передумови (Prerequisites)

\- Встановлений Terraform >= 1.5.0

\- AWS CLI налаштований (`aws configure`)

\- Git встановлений



\## Використання

1\. Ініціалізація Terraform:

terraform init



2\. Застосування конфігурацій:

terraform apply



3\. Видалення ресурсів:

terraform destroy
